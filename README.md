# focus.lab

A personal daily planner with two modules: Work and Life.

## Current functionality
- Main focus for Work and Life
- Regular tasks
- Optional target/count progress
- Reflection with previous-day gate
- Hover actions for complete/edit/delete

## Design rules
- White background
- Bluebird UI color: rgb(77,132,207)
- Chunky bold title
- Minimal desktop-first layout

## Goal
Turn this HTML prototype into a maintainable web app.
