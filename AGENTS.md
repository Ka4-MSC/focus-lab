# AGENTS.md

## Project goal
Build and improve focus.lab, a stylish daily planner web app.

## Product rules
- Keep the interface minimal and bold
- Preserve the white + bluebird visual identity
- Desktop-first, optimized for 13-inch laptop screens
- Do not add unnecessary features
- Prefer clean components and explicit state
- Avoid unnecessary dependencies

## Priorities
1. Stable task creation/editing/completion
2. Main focus logic for Work and Life
3. Reliable daily storage model
4. Future-ready architecture for backend sync
